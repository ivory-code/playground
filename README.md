## **🍎&nbsp;&nbsp; Run ios**

1. <code>yarn</code>
2. <code>cd ios && pod install // cd ..</code>
3. <code>yarn start --reset-cache</code>
4. <code>yarn ios</code>

## **🤖&nbsp;&nbsp; Run Android**

1. <code>yarn</code>
2. <code>cd android && ./gradlew clean // cd ..</code>
3. <code>yarn start --reset-cache</code>
4. <code>yarn android</code>
